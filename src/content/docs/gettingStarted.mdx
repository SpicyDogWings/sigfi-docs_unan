Esta es un manual de usuario en línea para los usuarios del sistema financiero Sigfi de la UNAN-León, con
el fin de que los usuarios siempre cuenten con la información del funcionamiento de los módulos de
dicho sistema.

Para más información de los módulos del sistema ir a la sección de <a href="/módulos/">Módulos</a>.

## ¿Porqué?

### Complejidad

El sistema financiero de la UNAN-León es muy complejo, tiene una gran cantidad de módulos y muchas veces puede ser confuso
enternder el porqué de un error que arroja el sistema o que posiblemente nos hace falta un paso para realizar
la acción que queremos ejercer. Para ello surge la idea de un manual que este siempre disponible para los usuarios
que les ayude a indentificar el error y facilitar la resolución del problema lo antes posible para no interferir 
con las tareas cotidianas de cada usuario.

### Inducción

No es misterio que cada empresa posee un método de trabajo distinto, en los nuevos intregrantes puede llegar a ser una tarea
complicada de entender o almenos tardada. Con esta idea los equipos podrán acceder facilmente al manual que se 
les dió accesso y consultar como funciona el sistema, o  más especifico, al módulos del sistema que les corresponde 
manipular a ell@s.
